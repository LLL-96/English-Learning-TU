# 📝 更新日志 - v2.1

## ✨ 新增功能

### 1. 🎤 英式/美式语音切换
- ✅ 新增语音类型选择器（自动/美式/英式）
- ✅ 根据语音偏好自动过滤可用语音列表
- ✅ 保存语音偏好到本地存储
- ✅ 所有朗读功能均支持语音切换

**使用方式**:
1. 打开设置面板（页面底部）
2. 找到"语音类型"下拉框
3. 选择"美式英语"或"英式英语"
4. 语音列表会自动更新

---

### 2. ✏️ 拼写练习模式
- ✅ 全新的拼写练习模式（5种学习模式之一）
- ✅ 听发音，拼写正确的单词
- ✅ 支持正常速度和慢速播放
- ✅ 智能提示系统（首字母 + 长度）
- ✅ 即时反馈（正确/错误）
- ✅ 练习结束后显示错误单词回顾
- ✅ 自动保存练习结果

**使用方式**:
1. 选择任意单元
2. 点击侧边栏的"拼写练习"按钮
3. 点击"▶️ 开始练习"
4. 听发音，输入单词拼写
5. 按 Enter 键提交

**特色功能**:
- 💡 **提示按钮**: 显示单词首字母和长度
- 🐌 **慢速播放**: 以 0.5 倍速播放发音
- 📊 **结果回顾**: 显示错误单词列表，方便复习

---

### 3. 📤 学习进度导出
- ✅ 支持导出学习进度为 JSON 文件
- ✅ 包含完整的学习统计数据
  - 学习时长
  - 已学单词数
  - 听写正确率
  - 测试平均分
- ✅ 导出收藏单词列表
- ✅ 记录导出时间和当前学习位置

**导出内容**:
```json
{
  "exportDate": "2026-04-07 15:30:00",
  "studyStats": {
    "studyTime": "45 分钟",
    "wordsLearned": "120 词",
    "dictationRate": "85%",
    "testAverage": "90 分"
  },
  "favorites": [...]
}
```

**使用方式**:
1. 点击顶部导航栏的"📊 统计"按钮
2. 查看学习统计数据
3. 点击"📤 导出学习进度"按钮
4. JSON 文件自动下载

---

### 4. 🌐 离线访问支持
- ✅ 新增 Service Worker 实现离线访问
- ✅ 缓存所有核心资源（HTML、JS、CSS、数据文件）
- ✅ 离线状态下仍可正常使用应用
- ✅ 自动更新缓存（版本升级时清理旧缓存）

**离线功能**:
- 首次访问后自动缓存所有资源
- 断网后仍可访问已缓存的内容
- 单词学习、课文朗读、听写、测试等功能全部可用
- 学习进度仍会保存到本地存储

**技术实现**:
- `service-worker.js`: Service Worker 脚本
- 缓存策略: Cache First, falling back to Network
- 缓存版本: `english-cs-v2.0`

## 🎯 使用建议

1. **首次使用**:
   - 建议在联网环境下首次访问，以便缓存所有资源
   - 选择适合的教材版本和语音类型

2. **日常学习**:
   - 按照"单词学习 → 课文朗读 → 拼写练习 → 单元测试"的顺序学习
   - 定期查看学习统计，了解自己的进度
   - 导出学习进度，备份重要数据

3. **离线使用**:
   - 首次访问后，即使断网也能正常使用
   - 建议定期联网更新缓存

## 💡 使用技巧

1. **拼写练习**:
   - 先用"慢速播放"熟悉发音
   - 再用正常速度练习拼写
   - 遇到难题点击"提示"查看首字母

2. **语音切换**:
   - 美式英语: 适合学习美音
   - 英式英语: 适合学习英音
   - 自动: 使用浏览器默认语音

3. **数据导出**:
   - 定期导出备份
   - 可用于跟踪学习进度
   - 支持导入到其他应用

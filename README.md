# 🎓 小学英语同步学习平台

一个功能完整的小学英语在线学习平台，支持三大教材版本，完全免费！

🌐 **在线访问**：https://lll-96.github.io/English-Learning-TU/

---

## ✨ 功能特点

- 📚 **三大版本**：人教版 PEP、外研社版、通用大纲版
- 🎯 **单元学习**：按单元组织，循序渐进
- 📖 **单词学习**：卡片视图 + 列表视图，含音标、例句
- 🔊 **课文朗读**：AI 语音朗读，支持全文连读
- ✍️ **单词听写**：智能听写，即时反馈
- ✏️ **拼写练习**：听音拼写，支持慢速播放和提示
- 📝 **单元测试**：10 道选择题，检验学习成果
- 🏷️ **标签分类**：按主题分类查看单词
- ⭐ **收藏功能**：收藏重点单词，方便复习
- 📊 **学习统计**：记录学习时长、正确率
- 📤 **进度导出**：导出学习进度为 JSON 文件
- 🌐 **离线访问**：Service Worker 支持，断网也能用
- 🎤 **语音切换**：支持美式/英式英语切换
- 🌙 **夜间模式**：保护视力，舒适学习
- 💾 **进度保存**：自动保存学习进度

## 💡 使用技巧

1. **选择教材版本**：根据孩子使用的教材选择对应版本
2. **单词学习**：先听发音，再看中文意思，最后看例句
3. **收藏重点**：将难记的单词加入收藏夹，重点复习
4. **定期测试**：完成单元学习后做测试，检验成果
5. **查看统计**：通过统计数据了解学习进度
